Из каких имён состоит доменное имя моего сайта

12 января 2026 г.

Доменное имя — это удобный текстовый адрес сайта, который заменяет сложный числовой IP-адрес. Оно устроено иерархически, начиная с самого общего уровня (справа) и заканчивая самым конкретным (слева).

Мой сайт размещён по адресу: https://dasha964.github.io

Разберём его структуру по уровням:

  • .io — домен верхнего уровня (TLD). Изначально относился к Британским Индийским океаническим территориям, но сейчас широко используется в IT-сообществе.
  • github — домен второго уровня. Это основное имя, зарегистрированное компанией GitHub.
  • dasha964 — поддомен (третий уровень), автоматически созданный для моего аккаунта на GitHub Pages.

Таким образом, полное доменное имя состоит из трёх частей, каждая из которых указывает на определённый уровень в глобальной системе DNS.
