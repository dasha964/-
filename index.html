Из каких имён и на каких уровнях состоит доменное имя моего сайта

Опубликовано: 12.01.2026 г.

Доменное имя — это человекочитаемый адрес сайта, который заменяет числовой IP-адрес. Оно устроено иерархически справа налево.

Мой сайт расположен по адресу: https://dasha964.github.io

Разберём его по уровням:

  • .io — домен верхнего уровня (TLD). Изначально национальная зона Британских Индийских океанических территорий, но сейчас популярна в IT-среде.
  • github — домен второго уровня. Это основное имя, зарегистрированное компанией GitHub.
  • dasha964 — поддомен третьего уровня. Он автоматически создаётся GitHub Pages для моего аккаунта.

Таким образом, полное доменное имя состоит из трёх частей, каждая из которых отвечает за определённый уровень в DNS-иерархии.
